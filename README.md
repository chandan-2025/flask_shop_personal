# Flask Shop Personal

A personal Flask project for managing shop-related operations, including admin dashboards, appointment booking, and status tracking.

## Project Structure

- `shop_flask.py` – Main Flask application file.
- `shop_templates/` – HTML templates used by the Flask app.
- `.gitignore` – To ignore virtual environment, caches, and other unnecessary files.

## Features

- Admin login and dashboard
- Book appointments
- Check appointment status
- Update shop settings
- Responsive HTML templates for shop management
